## React, Flask, TypeScript, and Webpack boilerplate

---
### Setup
1. Clone the Repo.
  ```sh
  git clone git@github.com:KandarpAjvalia/react-flask-boilerplate.git
  ```
2. Install dependencies.
  ```sh
  yarn install
  or
  npm install
  ```
3. Start the Flask server and Webpack at the same time.
  ```sh
  yarn start
  or
  npm start
  ```
